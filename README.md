# 🍅 TomatoGuard: Professional Tomato Leaf Disease Detector

TomatoGuard is a high-performance web application designed to help farmers and gardeners identify tomato leaf diseases instantly using state-of-the-art technology. The system employs a "Dual-Engine" architecture, featuring a primary cloud-based analysis and a robust local fallback engine.

## ✨ Key Features

*   **Dual-Engine Diagnosis**: Uses advanced vision technology for deep analysis and a robust fallback mechanism for disconnected or high-latency scenarios.
*   **Detailed Infection Insights**: Not only identifies the disease but explains **how it happens**, provides **natural treatments**, and suggests **recommended products**.
*   **Cinematic, Minimalist UI**: A premium, distraction-free interface built with React, Framer Motion, and Tailwind CSS.
*   **Capture or Upload**: Supports high-resolution camera capture or gallery uploads.
*   **History Tracking**: Keeps a local log of your recent plant health scans.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS 4.0
- **Animations**: motion/react (Framer Motion)
- **Icons**: Lucide React
- **Cloud Engine**: Google GenAI Technology

## 🚀 Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **Engine API Key**: Required for image processing.

### 1. Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Configure environment variables:
    Create a `.env` file in the root:
    ```env
    ENGINE_API_KEY=your_actual_key_here
    ```
4.  Run the development server:
    ```bash
    npm run dev
    ```
